Aritmética Computacional e Lógica Digital

01 - Sistema Binário

O computador, por ser uma máquina eletrônica, representa todas as informações através de dois níveis de tensão, que correspondem aos dígitos 0 e 1. Esse é o sistema binário, a linguagem fundamental da computação.

  • Bit e Byte: A menor unidade de informação é o bit (binary digit). Um conjunto de 8 bits forma um byte.
  • Linguagem de Máquina: Programas escritos em linguagens de programação (como Python ou Java) são "traduzidos" (compilados) para sequências de zeros e uns, que é a linguagem que o processador entende.
  • Álgebra Booleana: É a base matemática da computação, utilizando operações lógicas (E, OU, NÃO) que correspondem perfeitamente à lógica binária de verdadeiro/falso (1/0).
  • Codificação de Caracteres: Padrões como ASCII e Unicode usam sequências de bits para representar letras, números e símbolos.

02 - Sistemas de Numeração e Conversão de Bases

Além do binário (base 2) e do decimal (base 10) que usamos no dia a dia, a computação utiliza outras bases para representar números de forma mais compacta.

  • Bases Comuns:
    • Binário (Base 2): Usa os algarismos 0 e 1.
    • Octal (Base 8): Usa os algarismos de 0 a 7.
    • Hexadecimal (Base 16): Usa os algarismos de 0 a 9 e as letras de A a F (para representar 10 a 15).
  • Conversão de Bases: É o processo de transformar um número de uma base para outra.
    • Para Decimal: Multiplica-se cada dígito do número pela potência de sua base correspondente à sua posição e soma-se os resultados.
    • De Decimal: Realizam-se divisões sucessivas do número pela nova base, e o resultado é composto pelo último quociente e pelos restos das divisões, lidos de baixo para cima.
    • Entre Binário, Octal e Hexadecimal: A conversão pode ser feita diretamente agrupando-se os bits (3 para octal, 4 para hexadecimal).

03 - Aritmética Computacional

As operações aritméticas básicas (soma, subtração, multiplicação e divisão) podem ser realizadas diretamente nas bases binária, octal e hexadecimal, seguindo regras semelhantes às da base decimal.

  • Soma: Quando a soma dos dígitos em uma coluna atinge ou ultrapassa o valor da base, ocorre o "vai um" para a coluna seguinte.
  • Subtração: Utiliza o conceito de "pedir emprestado" da coluna à esquerda, onde o valor emprestado é igual ao valor da base.
  • Multiplicação e Divisão: Seguem lógicas análogas às da base decimal, mas podem ser complexas. Uma alternativa comum é converter os números para a base 10, realizar a operação e depois converter o resultado de volta para a base original.

04 - Portas Lógicas

As portas lógicas são os blocos de construção fundamentais dos circuitos digitais. São componentes eletrônicos que executam operações lógicas básicas da Álgebra Booleana.

  • Porta AND (E): A saída é 1 somente se todas as entradas forem 1.
  • Porta OR (OU): A saída é 1 se pelo menos uma das entradas for 1.
  • Porta NOT (NÃO): Inverte a entrada. Se entra 1, sai 0; se entra 0, sai 1.
  • Porta XOR (OU Exclusivo): A saída é 1 somente se as entradas forem diferentes.
  • Porta NAND (NÃO E): É o inverso da porta AND. A saída é 0 somente se todas as entradas forem 1.
  • Porta NOR (NÃO OU): É o inverso da porta OR. A saída é 1 somente se todas as entradas forem 0.

05 - Circuitos Lógicos

Circuitos lógicos são combinações de portas lógicas projetadas para executar funções específicas, desde somar números até selecionar dados.

  • Tipos de Circuitos:
    • Combinacional: A saída depende apenas das entradas atuais. Não possui memória (ex: uma calculadora).
    • Sequencial: A saída depende das entradas atuais e do estado anterior do circuito. Possui memória (ex: um contador).
  • Exemplos de Circuitos Combinacionais:
    • Somador Completo (Full Adder): Circuito que soma três bits (dois bits de entrada e um "vai um") e gera um resultado e um novo "vai um".
    • Decodificador: Converte uma entrada binária de 'n' bits em uma única saída ativa entre 2^n saídas possíveis.
    • Multiplexador (MUX): Funciona como um seletor, escolhendo uma de várias linhas de entrada para direcionar a uma única linha de saída.

Conclusão: Pontos Principais

  • A Base de Tudo: O sistema binário é a linguagem fundamental dos computadores, representando dados e instruções como sequências de 0s e 1s.
  • Múltiplas Representações: Sistemas como octal e hexadecimal são usados para representar dados binários de forma mais compacta e legível para humanos.
  • Lógica em Hardware: A Álgebra Booleana é implementada fisicamente através de portas lógicas, que são os componentes mais básicos de um processador.
  • Construindo Complexidade: As portas lógicas são combinadas para formar circuitos lógicos que executam desde operações aritméticas simples até tarefas complexas de processamento e controle.
  • Do Bit ao Circuito: Há uma progressão clara desde o conceito abstrato do bit, passando pela sua manipulação com aritmética e lógica, até a sua implementação física em circuitos que formam o coração de todos os dispositivos digitais.
