Linguagens e Algoritmos para Web

01 - HTML5: A Estrutura da Web

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para criar a estrutura e o conteúdo das páginas web. A versão atual, HTML5, introduziu tags semânticas que dão mais significado ao conteúdo.

  • Estrutura Básica: Todo documento HTML possui uma estrutura essencial com <!DOCTYPE html>, <html>, <head> (para metadados como o título) e <body> (para o conteúdo visível).
  • Tags Essenciais: Incluem títulos (<h1> a <h6>), parágrafos (<p>), links (<a>), imagens (<img>) e listas (<ul> para não ordenadas e <ol> para ordenadas).
  • Tags Semânticas: Elementos como <header>, <nav>, <main>, <section>, <article>, <aside> e <footer> ajudam a organizar a página de forma lógica, melhorando a acessibilidade e a otimização para motores de busca (SEO).

Exemplo de Estrutura Básica HTML:


<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <h1>Conteúdo Principal</h1>
    <p>Este é um parágrafo.</p>
</body>
</html>

02 - CSS3: A Estilização da Web

CSS (Cascading Style Sheets) é a linguagem utilizada para estilizar os elementos HTML, controlando cores, fontes, espaçamentos e o layout da página.

  • Sintaxe: Uma regra CSS é formada por um seletor (que aponta para o elemento HTML a ser estilizado) e um bloco de declarações com pares de propriedade: valor;.
  • Seletores Comuns: Os seletores podem ser por nome da tag (p), por classe (.minha-classe) ou por ID (#meu-id).
  • Formas de Inclusão: O CSS pode ser aplicado de forma externa (arquivo .css separado, a melhor prática), interna (dentro da tag <style> no <head>) ou inline (direto no atributo style de um elemento).
  • Box Model: Todo elemento HTML é tratado como uma caixa composta por quatro partes: o conteúdo, o padding (espaçamento interno), a border (borda) e a margin (espaçamento externo).

Exemplo de Sintaxe CSS:


/* Seletor de tag */
p {
  color: green;
  font-size: 16px;
}

/* Seletor de classe */
.destaque {
  background-color: yellow;
}

/* Seletor de ID */
#titulo-principal {
  text-align: center;
}

03 - Algoritmos e JavaScript: A Lógica da Web

JavaScript é a linguagem de programação que adiciona interatividade e comportamento dinâmico às páginas web. Sua lógica é baseada em algoritmos.

  • Algoritmo: É uma sequência de passos lógicos e finitos para resolver um problema. Em programação, o algoritmo é traduzido para uma linguagem que o computador entende.
  • Variáveis: São "recipientes" para armazenar dados. Podem ser declaradas com let (para valores que podem mudar e tem escopo de bloco), const (para valores constantes) ou var (valores que podem mudar e tem escopo de função. Forma mais antiga, menos recomendada).
  • Comandos de Entrada e Saída:
    • Saída: alert() exibe uma caixa de alerta, e console.log() escreve informações no console do navegador (útil para depuração).
    • Entrada: prompt() solicita que o usuário digite algo, e confirm() exibe uma caixa de diálogo de confirmação.
  • Operadores: Incluem os aritméticos (+, -, *, /), relacionais (==, >, <) e lógicos (&& para "E", || para "OU").
  • Manipulação do DOM: O JavaScript pode interagir com a estrutura do HTML (o DOM - Document Object Model) para alterar conteúdo, estilos e atributos em resposta a eventos, como um clique de botão (onclick).

Exemplo Prático de HTML e JavaScript (Lâmpada):

HTML:


<!DOCTYPE html>
<html lang="pt-br">
<head>
    <title>Exemplo Lâmpada</title>
</head>
<body>
    <h2>O que o JavaScript pode fazer?</h2>
    <button onclick="acendeLampada()">Acende Lâmpada</button>
    <img id="imagem" src="https://www.w3schools.com/js/pic_bulboff.gif" style="width:100px">
    <button onclick="apagaLampada()">Apaga Lâmpada</button>

    <script src="script.js"></script>
</body>
</html>

JavaScript (arquivo script.js):


function acendeLampada() {
  document.getElementById('imagem').src='https://www.w3schools.com/js/pic_bulbon.gif';
}

function apagaLampada() {
  document.getElementById('imagem').src='https://www.w3schools.com/js/pic_bulboff.gif';
}

Css (arquivo style.css):


function acendeLampada() {
  document.getElementById('imagem').src='https://www.w3schools.com/js/pic_bulbon.gif';
}

function apagaLampada() {
  document.getElementById('imagem').src='https://www.w3schools.com/js/pic_bulboff.gif';
}

Conclusão: Pontos Principais

  • Tríade do Desenvolvimento Web: HTML estrutura o conteúdo, CSS define a aparência e JavaScript controla o comportamento e a interatividade.
  • Semântica é Importante: Usar tags HTML semânticas torna o código mais claro, acessível e melhor para SEO.
  • Separação de Responsabilidades: A melhor prática é manter o HTML, o CSS e o JavaScript em arquivos separados para facilitar a manutenção e a reutilização do código.
  • Lógica Algorítmica: A base de toda interatividade criada com JavaScript é um algoritmo bem definido, que é a sequência de passos para resolver um problema.
  • DOM como Ponte: O JavaScript interage com o HTML através do DOM, permitindo a criação de páginas dinâmicas e responsivas às ações do usuário.
