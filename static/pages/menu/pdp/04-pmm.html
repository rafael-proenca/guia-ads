BOOTSTRAP

Uma CDN (Content Delivery Network), ou Rede de Distribuição de Conteúdo, é uma rede de servidores espalhados por diversas localidades geográficas. A CDN permite o uso do Bootstrap em nossa aplicação. Sendo assim, toda vez que a página for carregada, o servidor mais próximo da localidade da sua localidade se encarrega de fornecer o estilo da página.

CSS

Criaremos um arquivo externo de css para ser referenciado no nosso html, copiando o css e usando Media Query, que é um recurso do CSS3 que permite aplicar diferentes estilos a uma página web com base nas características do dispositivo em que ela está sendo exibida.
